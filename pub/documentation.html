Getting Started

To get started on using the library, you will first need to add the following script and stylesheet references to your webpage since SimpleForms.js does not rely on any other library. 

<script src="/path/to/simple-forms.js"></script>
<link rel="stylesheet" href="/path/to/simple-forms.css">

API Documentation

How to create forms

This section will show you how to initialize and create forms to begin adding items to.

Function Arguments Return Object Description
createForm formName Form Object Creates an empty form object with name formName to begin adding elements to.
createGrouping groupingName Grouping Object Creates an empty grouping object with name groupingName to begin adding elements to. Note that groupings must be appended to a form object to become visible when the form is added to the DOM.

How to add items to your forms

This section will show you how to add items to your forms so you can get the functionality you are looking for.

Whether you intialized a Form or a Grouping object, you can begin adding items to it using the following syntax: form.addItem(...args) or grouping.addItem(...args).

Function Arguments Description
addInputArea type: 'text' | 'email' | 'password', default: ' '
inputAreaPlaceholder: String, default: ' '
labelValue: String, defualt: ' '
validator: RegExp, default: null
Creates an input text area with type: type with placeholer text as inputAreaPlaceholder and top label value as labelValue and text validator as validator.
addCheckbox labelValue: String, default: ' ' labelSide: 'left', 'right', default: 'right' Creates a checkbox with a label of value: labelValue and on the side labelSide.
addSubmitButton buttonText: String, default: ' ', onClickFunction: function(event) Creates a button with text buttonText, when pressed executes onClickFunction.
addText text: String, default: ' ' Adds text to the form with value text.
addDropdown dropdownText: String, default: ' ' dropdownItems: Array[String] Creates a dropdown with dropdown button text as: dropdownText with dropdown options as dropdownItems.
addGrouping grouping: Grouping Object Adds grouping to the form.

Note that for SimpleForms.js to work, you must provide a styling for text to be able to restyle the form components. An example of this JS Object is provided below. 
        
          const styles = {
            form: {
              borderRadius: '8px',
            },
            text: {
              textSize: '16px'
            },
            inputArea: {
              borderRadius: '16px',
              inputHeight: '40px'
            },
            button: {
              backgroundColour: '#0b57bb',
              borderColour: 'black'
            }
          }

Displaying your form to the DOM

Once you have built up your form, you can add it to the DOM with one simple function.

Function Arguments Description
buildForm id: String
styles: JS Object 		Adds the form to the DOM at id id with styles given by styles.
